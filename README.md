# Rock Paper Scissors

A simple browser-based Rock Paper Scissors game built with HTML, CSS, and JavaScript.

## Files

- `index.html` - game layout and structure
- `style.css` - visual styling and responsive layout
- `main.js` - game logic, score tracking, and win state handling

## Features

- Player selects Rock, Paper, or Scissors
- Computer chooses a random move
- Displays winner for each round
- Tracks player and computer scores
- Shows a win state when either reaches 5 points
- Includes a restart button to reset the game

## How to run

1. Open `index.html` in a web browser.
2. Click one of the choice buttons: ✊, 🤚, or ✂️.
3. The game will show the result and update scores.
4. When a score reaches 5, a win screen appears.
5. Click `Restart Game` to play again.

## Notes

- The current `Play` button is present in the interface but does not have active functionality in the JavaScript code.
- The game uses emoji buttons for intuitive gameplay.
